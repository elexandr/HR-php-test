## Настройка проекта
Laravel:
- `composer install`
- настроить `.env` файл
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`

## Дополнительная информация

Для верстки использован bootstrap `/public/js/app.js`, `/public/css/app.css`
Свой js код в файле `/public/js/script.js` 
Свои css стили в файле `/public/css/style.css` 
Для работы погоды - добавить в `.env` в API_WEATHER_KEY= ключ разработчика Яндекс
